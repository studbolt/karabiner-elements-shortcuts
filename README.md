
# 🧠 Studbolt's Karabiner-Elements Shortcut Cheat Sheet

Core Modifier: **Caps Lock (⇪ held)** enables all mappings.

---

## 🚀 Navigation

| Key Combo       | Action                    |
|-----------------|---------------------------|
| ⇪ + i           | ↑ Arrow Up                |
| ⇪ + j           | ← Arrow Left              |
| ⇪ + k           | ↓ Arrow Down              |
| ⇪ + l           | → Arrow Right             |
| ⇪ + u           | ⌘← (Beginning of line)    |
| ⇪ + o           | ⌘→ (End of line)          |
| ⇪ + ⌘ + u       | ⌘↑ (Top of document)      |
| ⇪ + ⌘ + o       | ⌘↓ (Bottom of document)   |
| ⇪ + ⌘ + i       | ⌃p (Cursor up)            |
| ⇪ + ⌘ + k       | ⌃n (Cursor down)          |

---

## ✍️ Selection

| Key Combo           | Action                          |
|---------------------|---------------------------------|
| ⇪ + ⌘ + ⇧ + j       | ⇧⌘← (Select to line start)      |
| ⇪ + ⌘ + ⇧ + l       | ⇧⌘→ (Select to line end)        |
| ⇪ + ⌘ + ⇧ + u       | ⇧⌘↑ (Select to top of doc)      |
| ⇪ + ⌘ + ⇧ + o       | ⇧⌘↓ (Select to bottom of doc)   |
| ⇪ + ⌘ + ⇧ + j       | ⇧⌥← (Select word left)          |
| ⇪ + ⌘ + ⇧ + l       | ⇧⌥→ (Select word right)         |

---

## 🧹 Deletion

| Key Combo       | Action                            |
|-----------------|-----------------------------------|
| ⇪ + n           | ⌫ (Backspace)                     |
| ⇪ + p           | ⌦ (Forward delete)                |
| ⇪ + ⌘ + n       | ⌥⌫ (Delete word left)             |
| ⇪ + ⌘ + p       | ⌥⌦ (Delete word right)            |
| ⇪ + ⌥ + n       | ⌘⌫ (Delete to line start)         |
| ⇪ + ⌥ + p       | ⌘⌦ (Delete to line end)           |

---

## 📄 Paging

| Key Combo       | Action               |
|-----------------|----------------------|
| ⇪ + y           | Page Up (fn ↑)       |
| ⇪ + h           | Page Down (fn ↓)     |
| ⇪ + ⌘ + y       | Page Up (fn ↑)       |
| ⇪ + ⌘ + h       | Page Down (fn ↓)     |

---

| 🌐`             | ⌃F2 (Focus Menu Bar)            |

| ⇧⌫              | ⌦ (Forward delete)              |
| ⌘⌫              | ⌥⌫ (Delete word, ⇪ required)    |

## 🪟 Window / Layout

|-----------------|--------------------------------------|

## 🧷 Legend

⇪ = Caps Lock (held) &nbsp; • &nbsp; ⌘ = Command &nbsp; • &nbsp; ⌥ = Option &nbsp; • &nbsp; ⌃ = Control &nbsp; • &nbsp; ⇧ = Shift &nbsp; • &nbsp; ⌫ = Backspace &nbsp; • &nbsp; ⌦ = Delete
